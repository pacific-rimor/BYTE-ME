# Smart City Dashboard

A comprehensive web application for monitoring and analyzing urban systems for sustainable, healthy, and agriculturally-supported smart cities.

## Features

### 5 Integrated Modules:
1. **Urban Mobility & Traffic System** - Real-time traffic monitoring and congestion management
2. **Energy & Environmental Monitoring** - Air quality, energy consumption, and environmental metrics
3. **Public Health Analytics** - Pollution-related health risks and hospital load monitoring
4. **Agriculture & Food Supply Monitoring** - Crop yield, water efficiency, and food stability
5. **Integrated Smart City Dashboard** - Main dashboard with cross-module analytics

### Key Capabilities:
- Real-time data simulation and updates (every 5 seconds)
- Rule-based predictive analytics
- Interactive scenario planning (what-if analysis)
- Automated alerts and recommendations
- Responsive Bootstrap 5 interface
- Chart.js visualizations
